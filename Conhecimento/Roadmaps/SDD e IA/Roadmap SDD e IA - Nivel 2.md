# Roadmap SDD e IA - Nível 2
## Meta: Integrar IA em requisitos e criar especificações completas

**Duração**: 8 semanas
**Pré-requisito**: Ter completado o Nível 1 com 70% ou mais no questionário

---

## Estrutura Semanal

### Semana 9-10: Arquitetura de IA com SDD
- [ ] estudar padrões de integração de IA em software
- [ ] desenhar arquitetura de alto nível para uma feature com IA (escrever à mão)
- [ ] listar requisitos técnicos para integração (APIs, custos, segurança)
- [ ] criar requisitos de governança de dados para IA
- [ ] escrever documento de especificação técnica completo
- [ ] revisar e validar com checklist de qualidade

### Semana 11-12: Prompt Engineering e Requisitos
- [ ] aprender princípios de prompt engineering
- [ ] criar biblioteca de prompts para diferentes cenários (escrita à mão)
- [ ] definir requisitos de prompts (temperatura, tokens, etc.)
- [ ] criar testes de prompts (escrever à mão o que testar)
- [ ] documentar casos de uso de prompts no projeto
- [ ] fazer autoavaliação de qualidade de prompts

### Semana 13-14: Testes e Validação de IA
- [ ] definir critérios de aceitação para features com IA
- [ ] criar matriz de testes para IA (latência, qualidade, usabilidade)
- [ ] escrever plano de testes para feature com IA
- [ ] executar testes em protótipo (se disponível)
- [ ] documentar resultados e ajustes necessários
- [ ] criar checklist de validação de IA

### Semana 15-16: Observabilidade e Entrega
- [ ] definir métricas de monitoramento para IA
- [ ] criar dashboard de observabilidade (escrever o que incluir)
- [ ] estimar custos de API de IA (escrever cálculo)
- [ ] definir políticas de uso e governança
- [ ] preparar apresentação de caso de uso completo
- [ ] fazer avaliação final do roadmap

---

## Questionário de Avaliação - Nível 2

Responda por escrito:

1. Descreva a arquitetura de uma feature com IA que você especificou.
2. Liste 5 prompts que você criou e explique o uso de cada um.
3. Quais são os critérios de aceitação para uma feature com IA?
4. Quais métricas você definiu para monitorar a qualidade da IA?
5. Quais são os custos estimados e como você os calculou?

**Meta mínima**: Acertar 75% para conclusão do roadmap

---

## Conclusão do Roadmap SDD e IA

Ao completar este roadmap, você terá:
- Capacidade de escrever requisitos completos com IA
- Entendimento de arquitetura e integração de IA
- Habilidade de criar prompts e testes para IA
- Base para implementar IA em projetos reais
