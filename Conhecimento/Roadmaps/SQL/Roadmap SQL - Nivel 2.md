# Roadmap SQL - Nível 2
## Meta: Atingir nível intermediário-avançado em SQL

**Duração**: 6 semanas
**Pré-requisito**: Ter completado o Nível 1 com 70% ou mais no questionário

---

## Estrutura Semanal

### Semana 7-8: Funções Avançadas e Window Functions
- [ ] aprender e praticar window functions (ROW_NUMBER, RANK, DENSE_RANK)
- [ ] escrever consultas com PARTITION BY
- [ ] praticar funções de data (DATE, DATETIME, EXTRACT)
- [ ] resolver 5 desafios avançados de window functions no Codewars
- [ ] criar flashcards de window functions
- [ ] fazer exercícios de complexidade alta

### Semana 9-10: Otimização e Performance
- [ ] entender índices e como eles funcionam (escrever resumo à mão)
- [ ] aprender sobre EXPLAIN/EXPLAIN ANALYZE
- [ ] identificar consultas lentas e otimizar (escrever 5 exemplos)
- [ ] praticar normalization e evitar dados redundantes
- [ ] criar checklist de boas práticas de performance
- [ ] resolver desafios de otimização no Codewars

### Semana 11-12: Projeto Prático e Consolidação
- [ ] criar schema de banco de dados para projeto (escrever design)
- [ ] escrever consultas complexas para o projeto (10+ consultas)
- [ ] implementar views e stored procedures básicas
- [ ] documentar todas as consultas criadas
- [ ] fazer autoavaliação: conseguir resolver 10 desafios difíceis no Codewars
- [ ] revisar todo o conteúdo do roadmap

---

## Questionário de Avaliação - Nível 2

Responda por escrito:

1. Para que serve ROW_NUMBER() OVER (PARTITION BY...)? Dê um exemplo.
2. Quais são as boas práticas para criar índices? Liste pelo menos 5.
3. Como você otimizaria uma consulta que está lenta? Passos a seguir.
4. Escreva uma consulta que use window function para ranquear vendas por mês.
5. Quais são as diferenças entre UNION, UNION ALL, INTERSECT, EXCEPT?

**Meta mínima**: Acertar 75% para conclusão do roadmap

---

## Conclusão do Roadmap SQL

Ao completar este roadmap, você terá:
- Domínio de consultas SQL intermediárias e avançadas
- Conhecimento de window functions
- Habilidade de otimizar consultas
- Base sólida para trabalhar com bancos de dados em projetos
