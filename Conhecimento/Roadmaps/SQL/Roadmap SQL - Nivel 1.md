# Roadmap SQL - Nível 1
## Meta: Dominar consultas SQL básicas e intermediárias

**Duração**: 6 semanas

---

## Estrutura Semanal

### Semana 1-2: Fundamentos de SELECT e Filtros
- [ ] revisar conceitos básicos de SQL (SELECT, FROM, WHERE)
- [ ] escrever à mão 20 comandos SELECT básicos
- [ ] praticar no Codewars: 5 desafios de SELECT básico
- [ ] criar lista de comandos SQL básicos (escrita à mão no Samsung Notes)
- [ ] escrever consultas usando DISTINCT, ORDER BY, LIMIT
- [ ] fazer quiz de autoavaliação sobre SELECT

### Semana 3-4: Filtros e Operadores
- [ ] escrever consultas com operadores (AND, OR, NOT)
- [ ] praticar LIKE, IN, BETWEEN, IS NULL
- [ ] resolver 5 desafios no Codewars de filtros
- [ ] criar consultas com funções de agregação (COUNT, SUM, AVG, MIN, MAX)
- [ ] escrever à mão exemplos de cada função de agregação
- [ ] fazer exercícios de GROUP BY

### Semana 5-6: Junções e Subconsultas
- [ ] entender e praticar INNER JOIN (escrever 10 exemplos)
- [ ] praticar LEFT/RIGHT JOIN e FULL OUTER JOIN
- [ ] resolver 5 desafios de JOIN no Codewars
- [ ] escrever subconsultas (subqueries) básicas
- [ ] criar consultas com CASE WHEN
- [ ] fazer revisão geral e autoavaliação

---

## Questionário de Avaliação - Nível 1

Responda por escrito (pode usar português ou pseudocódigo):

1. Escreva uma consulta que retorne o total de vendas por cliente
2. Qual a diferença entre INNER JOIN e LEFT JOIN? Dê um exemplo.
3. Como usar LIKE para encontrar nomes que começam com "A"?
4. Escreva uma subconsulta que encontre os 5 produtos mais vendidos
5. Para que serve GROUP BY? Dê um exemplo.

**Meta mínima**: Acertar 70% para avançar ao Nível 2
