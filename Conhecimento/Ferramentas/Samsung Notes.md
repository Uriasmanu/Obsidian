# Samsung Notes

Ferramenta de anotações usada para estudo.

## Como uso para estudo

- Escrever conceitos à mão para que se fixem na memória
- Criar flashcards de vocabulário
- Anotarphraseusefulseexpressões novas
- Fazer diagramas e esboços
- Escrever diários de estudo

## Dicas de Uso

- Usar caneta stylus para melhor precisão
- Criar cadernos separados por assunto (SDD, IA, Inglês, Jiu-Jitsu)
- Sincronizar com Samsung Cloud para acessar em outros dispositivos
- Exportar notas como PDF para revisão
