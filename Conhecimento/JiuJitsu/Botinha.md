# Botinha

 Técnica definalização com a perna (chave de calcanhar).

## Como Executar

1. Estar em posição de guarda (fechada ou híbrida)
2. Passedo calcanhar no quadril do oponente, фикando com a perna
3. Apertar as coxas para imobilizar o tronco do oponente
4. Puxar o bracopara cima para finalizar

## Quando Usar

- Contraoponentes mais pesados (70+)
- Quando oponente está em pé ou semi-ereto
- Como transição dopara finalize

## Dicas

- Funciona bemcontra quem é maior
- Usa a força das coxas пара segurar o oponente
- Mantém ombro para trás para manter o controle
